# Meu Primeiro Repositório

Bem-vindo ao meu primeiro repositório git! 🎉

Este é um projeto de aprendizado para praticar com git e GitHub.

## 🚀 Como começar

1. Clone este repositório:
   ```bash
   git clone https://github.com/willianmlouzada/my-first-repo.git
   cd my-first-repo
   ```

2. Crie uma branch para sua feature:
   ```bash
   git checkout -b minha-feature
   ```

3. Faça suas alterações e commit:
   ```bash
   git add .
   git commit -m "Descrição clara da mudança"
   ```

4. Faça push e abra um PR:
   ```bash
   git push origin minha-feature
   ```

## ✅ Checklist de Aprendizado

- [x] Conectar ao GitHub
- [x] Criar primeira branch
- [x] Fazer primeiro PR
